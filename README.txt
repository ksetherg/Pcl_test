# demo
Особенности реализации https://github.com/PointCloudLibrary/data/tree/master/biwi_face_databaseпрограммы:
-принимает с коммандной строки имя исходного облака и имя файла, куда запишется преобразованное облако (расширение файлов .pcd)
-внутри программа добавляет гауссовский шум к исходному и сохраняет в указанный файл
-считает метрики: абсолютную разницу между облаками и среднюю разницу и выводит их в терминал
использовалась библиотека PCL, данные взял отсюда https://github.com/PointCloudLibrary/data/tree/master/biwi_face_database
рекомендации по компиляции:
-перейти в папку build
-собрать $cmake ../
-скомпилировать $make
-запустить $./pcl-test
